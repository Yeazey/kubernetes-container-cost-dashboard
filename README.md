# Kubernetes Container Cost Dashboard

Interactive Kubernetes Container Cost Dashboard powered by Cloudability MCP. Visualizes cluster spend, namespace costs, idle resources, and optimization opportunities across multi-cloud K8s environments.

## Features

- **4-tab dashboard**: Overview, Top Clusters, Optimization, KPIs
- **Multi-cloud support**: AWS EKS, OCI OKE, Azure AKS
- **Idle resource detection**: Highlights IDLE RESOURCES namespaces for savings opportunities
- **Weekly trends**: 13-week spend trends for top clusters
- **Self-contained HTML**: Single file, no server needed, uses Chart.js via CDN

## How It Works

This dashboard is generated by an AI agent (Kiro CLI) that:
1. Pulls live container cost data from Cloudability via MCP tools
2. Queries `container_cluster_name` and `container_namespace` dimensions
3. Calculates idle vs utilized costs per cluster
4. Generates a self-contained HTML dashboard with Chart.js visualizations

## Kiro Skill

Copy the `SKILL.md` into your `.kiro/skills/containersdash/` directory to enable the skill in Kiro CLI. Then just ask for "container dashboard" or "k8s costs".

## Data Sources

Uses Cloudability cost reporting API with container dimensions:
- `container_cluster_name` — Kubernetes cluster name
- `container_namespace` — Kubernetes namespace
- Metrics: `unblended_cost`, `total_amortized_cost`
- Idle detection via namespace = "IDLE RESOURCES"

## Cluster Naming Convention

- `*-akp-*` = AWS EKS
- `*-oke-*` = OCI OKE
- Prefix decoding: `uw2`=us-west-2, `ew1`=eu-west-1, `ec1`=eu-central-1, `ase2`=ap-southeast-2, `ash1`=us-ashburn-1, `frk1`=eu-frankfurt-1
